Custom levels:
To use custom backgrounds
	make sure the image is a 1280x720 png
	visible level image is within the middle, 400 pixels from the left, 480x720
		I suggest you take your full image and resize it to 480x720
		then go into GIMP or something and place it 400 pixels from the left so that it sits in the middle
		make sure the rest of the image is transparent and save as a png
	(you can use 200x720 on either side to make custom borders)
Custom mirror levels currently not supported.
